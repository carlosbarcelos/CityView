SURV / MQP Information

The SURV system is entirely hosted on a website and runs off of simple HTML and JavaScript. The data visualization is a product of applying CSV files of spatial-temporal data to a map provided by CARTO, MapBox, or Leaflet. The purpose of the project is to demonstrate how data visualization can be used for urban development planning in the real-world as a better option than simple data on a page.

Site Features Information

Home Page

The home page provides a short video demonstration of the project with some visualization examples.

About Page

The about page gives an explanation of both the project as a whole and the layout of the website. In addition, here you can find information about the students working on the project and their advisors.

Data Information

The data information page explains the different types of data presented in this project, and provides information about each type for those who may be newer to this field. The data information page also allows any user to download the datasets presented in this project to use for their own interested.

Data Examples

Here, a user can interact with the various visualizations created in the project to see what each data type looks like.
