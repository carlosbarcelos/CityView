On Spatial Temporal Datasets

Urban data can be catagorized by three combinations of spatial-temporal data. Spatial refering to the location of the data point, generally defined by an ordered pair: (latitude, longitude). Temporal refering to the time which the data point was collected, often in the form: yyyy-mm-ddThh:mm:ss

Please note, that the datasets whose combination is spatially dynamic and temporally static is impossible to collect as this type of data implies instantaneous change which is not feasible for physical data sets.

These are the data sets we used and information about them.

Spatially Static and Temporally Static

This type of data refer to fixed points in space. When applied to a map they reveal information about geographical distribution of some dataset.

These are the spatially statics and temporally static data sets we used and information about them.

  • Streetlight Locations
    • Boston provides information on the street lights in the city. The CSV containing this information contains 74065 street lights listed by their latitude and longitude.

Spatially Static and Temporally Dynamic

This type of data shows the change in a particular location over time. This can refer to the change of a number of catagories of data, often relating to the collection or redistribution of some entity at a location.

These are the spatially static and temporally dynamic data sets we used and information about them.

  • Trash Schedule
    • Trash pickup occurs in specific locations in the city on a timely basis. Trash pickup can be used to observe traffic patterns and determine bottlenecks.

Spatially Dynamic and Temporally Static

This type of data is impossible due to the fact that it would require instantaniou change.

Spatially Dynamic and Temporally Dynamic

This type of data shows how an object moves over time. By linking together many datapoints for a single object with unique time and location values, it possible to view change over time. The most appropraite analogy for this behavior is how an object travels.

These are the spatially dynamic and temporally dynamic data sets we used and information about them.

  • Boston Hubway Bikeshare
    • Bike share information lists the times and locations for each pickup and dropoff of the Boston Hubway bike share system. With this data it is possible to track how bikes move around the city.
