On Spatial-Temporal Datasets

Urban data can be categorized into three combinations of spatial-temporal data. Spatial refers to the location of the data point, generally defined by an ordered pair: (latitude, longitude). Temporal refers to the time at which the data point was collected, often in the form: yyyy-mm-ddThh:mm:ss

Please note that datasets with the combination of spatially-dynamic and temporally-static are impossible to collect, as this type of data implies instantaneous change, which is not feasible for physical data sets.

These are the data sets we used and information about them.

Static Data

This type of data refers to fixed points in space. When applied to a map they reveal information about geographical distribution of some dataset. It can be described as both spatially-static and temporally-static.

These are the static data sets we used and some information describing the datasets.

  • Streetlight Locations
    • Boston provides information on the street lights in the city. The CSV containing this information contains 74065 street lights listed by their latitude and longitude.

Time Series Data

This type of data shows the change in a particular location over time. This can refer to the change of a number of catagories of data, often relating to the collection or redistribution of some entity at a location. It can be described as spatially-static and temporally-dynamic.

These are the time series data sets we used and some information describing the datasets.

  • Trash Schedule
    • Trash pickup occurs in specific locations in the city on a timely basis. Trash pickup can be used to observe traffic patterns and determine bottlenecks.

Spatially Dynamic and Temporally Static

This type of data is impossible due to the fact that it would require instantaniou change.

Dynamic Data

This type of data shows how an object moves over time. By linking together many datapoints for a single object with unique time and location values, it possible to view change over time. The most appropraite analogy for this behavior is how an object travels. It can be described as both spatially-dyanamic and temporally-dyanamic.

These are the dynamic data sets we used and some information describing the datasets.

  • Boston Hubway Bikeshare
    • Bike share information lists the times and locations for each pickup and dropoff of the Boston Hubway bike share system. With this data it is possible to track how bikes move around the city.
